Nieuwe Installatie van IntegraMOD
Wat te doen voor en tijdens de Installallatie

Afkortingen die in deze documentatie worden gebruikt:

xxxxxx = Jokertekens, in de meeste besturingssystemen wordt hiervoor het "*" gebruikt
CHMOD = commandonaam om bestands rechten te veranderen
www.yoursite.com = moet met de naam van uw website worden vervangen
ACP = Admin Controle Paneel (Beheer Pagina's)

Vereisten:

Om IntegraMOD op uw server te installeren, hebt u nodig...

1) Een database, welke overeenkomt met MySQL3 of MySQL4
Een bewerkingsprogramma zoals PHPMyAdmin kan nuttig zijn, maar is niet essentieel.

2) Een host server met PHP Scripting 4.x/5.x

3) Schijfruimte op de server van minstens 50Mb
U heeft meer ruimte nodig wanneer u gebruik wilt gaan maken van de volgende mogelijkheden:
- Meerdere stijlen
- Bestanden opslaan
- Foto's
- Attachments

Installatie:

Voor degenen die vorige versies van IntegraMOD gebruikt hebben: Jullie zullen merken dat het installatie programma sterk verbeterd is. Voor nieuwe gebruikers van IntegraMOD: Welkom bij het beste Pre-modded phpBB pakket dat beschikbaar is.

1) Als onderdeel van het verbeterde installatie proces hebben we getracht de upload en permissie instellingen zo eenvoudig mogelijk te houden. Om maximaal gebruik te kunnen maken van de installatie vereenvoudigingen, moet je weten welk type server systeem je hebt.

Voor WINDOWS Servers - Bij de meeste windows servers kun je geen files unzippen op de server; ze hebben echter ook een andere manier waarop ze rechten behandelen wat een deel van de installatie vereenvoudigt.
- Voer taak a en b uit en ga vervolgens verder naar Taak 2

Voor UNIX/LINUX based servers die geen zip-files kunnen uitpakken - met deze servers moet je tijdens de installatie het meeste doen, waarbij de bulk van het werk ligt in het uploaden van de files via je ftp-client en vervolgens de onderstaande CHMOD's handmatig toepassen. Wanhoop echter niet, je inspanningen zullen worden beloond.
- Voer taak a, b en e uit en ga vervolgens verder naar Taak 2

Voor UNIX/LINUX based servers die wel zip-files kunnen uitpakken - nieuwere versies van deze servers hebben hun eigen file managers en controle panelen. De meest populaire daarvan is CPanel. Met gebruikmaking van onderstaande instructies kun je de file manager gebruiken om de zip te uploaden en uit te pakken. De CHMOD rechten zullen dan automatisch juist ingesteld zijn.
- Voer taak a, c en d uit en ga vervolgens verder naar Taak 2

a) Download de meest recents IntegraMOD installatie zip-file vanaf de IntegraMOD website of vanaf SourceForge.

b) Unzip de file naar een map op je lokale PC en Upload de files vervolgens met een FTP programma naar de map "public_html" of een subdirectory (bijv. /forum/) op je webserver.

c) Upload de volledige Installatie zip-file met een FTP programma of met de File Manager naar de map "public_html" of een subdirectory (bijv. /forum/) op je webserver. Unzip hem NIET voordat je hem geupload heb naar je server!

d) Unzip de Installatie zip-file via de File Manager of via de command prompt. Wanneer het systeem dit toestaat, zullen alle CHMOD instellingen intact zijn. Je kunt ze controleren met behulp van de instellingen die beschreven staan in sectie e. KLIK HIER voor meer informatie hoe je unzip gebruikt met CPanel.

e) Geef de onderstaande files en mappen op je server met behulp van een FTP programma of File Manager de volgende CHMOD rechten;

Unzip met CPanel!

- Om het leven gemakkelijker te maken, Open CPanel van je domein, en klik op File manager.
- Dan zie je je server.
- Ga naar de public_html directory of /forum_folder/ die je wil gebruiken.
- Klik op upload file, en selecteer je IntegraMOD pre-zipped download.
- Je zult het uploaden veel sneller en gemakkelijker vinden, omdat je met maar 1 bestand te maken hebt.
- Wanneer hij geupload is zul je het zip bestand op het scherm zien. Als je hem niet ziet, vernieuw dan de pagina.
- Klik op de tekst naam van het zip bestand en je ziet aan de rechter kant een lijst inclusief 'Extract folder contents'.
- Klik daar op, en er opent een nieuw scherm en ziet dat het bestand in seconden uitgepakt wordt.
- Aangezien bestand en directorie rechten al in het zip bestand zaten zouden ze met het uitpakken gezet moeten zijn.

2) Maak een lege Mysql Database aan

3) Maak een MySql Database Gebruiker en wijs een password toe

4) Voeg een gebruiker toe aan de Database met ALLE rechten

5) Gebruik je webbrowser voor de volgende taken:
Ga naar de locatie waarnaar je je files heb geupload en voeg daar "install/install.php" aan toe
(zonder aanhalingstekens)
bijv. http://www.jouwsite.nl/forum/install/install.php

6) Vul de benodigde informatie op de installatie pagina in en klik op "Start Installatie"
- (Overtuig jezelf ervan dat je ingevoerde informatie correct is - verkeerde informatie kan ertoe leiden dat je forum niet toegankelijk is of dat er niets zichtbaar is wanneer je wel toegang hebt).
- Op het scherm zul je een voortgangspagina zien terwijl je database wordt geïnstalleerd. Aan het eind zul je een balk zien waarom je kunt klikken om de installatie te completeren. KLIK HIER NIET OP (Taak 9) voordat je Taak 8 hebt gedaan.

7) Verwijder de "install" directory van je server.
Geef de directory niet slechts een andere naam. Het is verplicht deze directory geheel te verwijderen.

8) Nu je de installatie directory hebt verwijderd, klik je op 'Ik heb de installatie directory verwijderd... Installeer nu de Prillian...'

9) Je zult zien dat de installatie voor de Prillian gaat lopen. Normaliter is dit een kwestie van rechtsonder op de 'Ga naar de Volgende Stap' linken klikken. Volg de stappen voor installatie van de nieuwe Prillian Messenger.

10) Wanneer je de knop Prillian Installatie Gereed ziet, verwijder je de "prill_install" directory van je server.
Geef de directory niet slechts een andere naam. Het is verplicht deze directory geheel te verwijderen.
Op dit punt aangekomen moet je tevens de CHMOD permissies van de files "config.php" EN "includes/phpbb_security.php" wijzigen zodat ze enkel door jouw beschreven kunnen worden (CHMOD 664).

11) Klik nu op de 'Ik heb de prill_install directory verwijderd... Completeer de overgebleven registratie details voor mijn account' link. Nadat de Prillian installatie gereed is, zul je zien dat je de vereiste velden in je eigen profiel moet invullen.

12) Velden met een rood sterretje erachten MOETEN ingevuld worden.

13) Klik op "Verstuur" aan de onderzijde van de pagina. Je profiel is nu bijgewerkt. Klik op de "Home" link aan de bovenzijde van de pagina en je zult nu je forum zien.

14) Mocht je uitgelogd worden van je Forum, dan kun je inloggen door de Administrator details te gebruiken die je op het allereerste installatie scherm hebt ingevuld.

Gefeliciteerd! Je hebt het eerste deel van de installatie procedure doorlopen. Je zult op het Administratie Controle Paneel instellingen moeten wijzigen om je forum in te stellen zoals jij dat wilt.

Nu heb je een goede installatie, Ga naar Pagina 2 voor meer instellingen.
