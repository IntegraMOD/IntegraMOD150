Fresh Install for IntegraMOD 1.4.1 Page 2
Settings to do after Installation

Enter the Admin Control Panel (ACP):

1) Click on Admin in the menu bar (or "Go to Administration Panel" at the bottom of the page)

2) It will ask you for your log in details again. You need to fill this in every session that you go to the ACP

You are now ready to configure your system

Basic ACP Layout:

You will notice the main column on the right contains Forum Statistics, IntegraMOD News which is linked to e-tegra.com. to keep you informed of any important information, and Version Information, containing phpBB update information. Below this is a list of who is online and the IP address from where they have accessed the forum.

On the left hand column is a navigation menu to get you to settings within 2 clicks.

The following are settings to personalise your forum and make sure it performs as you would hope. I have placed a short comment about each section. On every page you need to submit or update these setting you change. If you change a variable and then move on without submitting, it will not save the changed contents.
 

Administration
Here are five links that allow you to navigate the main part of your forum. Previewing Forum and Portal will maintain the Admin Navigation column for easy access back to the ACP

Security

This is the an important addition to IntegraMOD and whilst it may occasionally frustrate, most of the time it will help protect you.
Configuration - Login Attempts - initially set at three but if you have a forum with noobs and less computer able, you may want to up this to 10
- Notify Admin- you may wish to be notified by only one of these constants
Special - Protect admin & moderator accounts -
If set to enabled you will not be able to have more than the specified number of admins (Amount of allowed admins) moderators (Amount of allowed mods) on your forum. If you intend to have 2 or more admins or moderatos you must set the specific numbers or disable this constant.

- Amount of allowed admins & Amount of allowed mods as explained above.
Redirect Admin PM's
If you have an admin that normally receives all PMs or you have a dummy admin user you can block or redirect PMs for Admin staff with this tool.
Attachments

This allows you to control items attached to posts within your forums

Approval
Here if you have set restrictions on users or forums, you can see all who wish approval and you can administer from this panel.
Download
This Download manager allows you to manages downloads from your site or elsewhere. Setup your download limits and banned extensions in here.
Extensions

File extension allowed and forbidden are named and configured here

eXtreme Styles
This is the area you can manage the style and templates for your site. It is very important part of personalising your forum. Her you can make sure everyone sees the style you want them too. You can also clone a style then develop it into your own.
Configuration - Show on left frame - You can select what items to show on left frame in admin control panel. You may find selecting Default Style and Manage Cache useful to select here.
- Default template directory - check initially that it says integra2
- make sure that "set Add tpl filenames in html" is set to No.
Styles Management - Set default style - switch all users to use an IntegraMOD 1.4.x compatible style.
NOTE: It is recommended to uninstall all those styles that are NOT IntegraMOD 1.4.x compatible.
- Manage Cache - Click 'Clear All', wait for next page to load, then at the bottom of the page, click on 'Compile All'. It may say it could take some time, at this stage, it won't :)
FTR Management
This is the Force To Read Topic, where you can make sure that every registered user reads the specified topic before they read anything more. It is initially set ON but you can turn it of or amend the message for new users.
FAQ Admin
You have an opportunity here to amend, and develop FAQ pages that already are set up, if you wish.
Forum Admin
this is the heart of how you structure your forums. You can set permissions for access, make categories, and manage whether points are added for posting. There is already a demo forum provided but it would be better in time to remove this. (Please Note the FTR topic is initially set to the welcome post in this forum. You will need to change the FTR configuration if you delete this post or the test forum.)
Spellcheck One of the important tools for your forum is the spell checker. Build your dictionaries in this section. Remember part1 then part2 of each language you wish to support. You must install at least one dictionary in order to use the Spell Check option. If no dictionary is installed the spell check popup window will appear blank.
NOTE: The Dictionaries take up a lot of DB Space. (approx 4-5mb each language). Delete the /spelling/xxxxxx.dic Files once you've built your Dictionaries.
General Admin
This is the nerve centre of the ACP. There is much in this section and will take time to feel fully confident. The more familiarity you gain of this section, the more solid your forum will be.
phpBBMyAdmin This is a phpBB's own cut down version of phpmyadmin. You will find this useful if you wish to quickly look up tables and manage elements of your forum.
Acronyms If you have words specific to your usergroup or abbreviations that may be less known, place them in this section and they will be highlighted in the forum to provide a reference to all.
Auto language detection This very useful feature detects the language set in a browsers configuration, so that when an unregistered user accesses your site, thay are greeted and can register details in their own language. It is worth noting that these need to be set when a new language is added to your forum.
Backup Database Although not essential, it is worth betting into the habit of backing up your database, in case of server problems or site attack. this section will allow the Admins the ability to backup the database on their local computer.
Configuration VERY IMPORTANT section
- Site Name - This will appear at the top of your browser windows and is the name of your portal. IntegraMOD adds the term 'Forum' after this text as the title of your system.
- Description - Go on, sell yourself. This is where you detail what you site is about and who it is aimed at.
- Disable board - DO NOT click Yes to this and log out. You will find it difficult to log back in if the board is disabled.
- Enable account activation - This is to help moderate who joins your board. It may be wise to initially set this as 'Admin', and it means you will need to approve anyone who joins. Once you are happy that your e-mail from the board is functioning properly, you will generally want to select 'User' here. This means this user will need to verify there e-mail address when they join. Unless you wish a completely anonymous Forum, or you are having mailing difficulty it would be preferable not to keep this set at 'None'
- User email via board - If selected 'Yes' then users will be able to mail via an e-mail messaging page, in order to protect that mail address from anyone other than the owner. If No is selected, the e-mail button holds the address and is linked to open the sender's mail program for a new mail on clicking the e-mail button.
- Default Language - This dropdown will only show installed languages.
- System Timezone - This is the Timezone your Host server is based in.
- Logo Setting - you are able to change aspects of your main logo here.
- Cookie name - If you have had another phpbb type forum before on your domain, it is highly recommended that you change the cookie name to different one as it can conflict when previous users of older forums come to visit your new site.
- Avatar Settings - IntegraMOD now has a set of avatars within your install. You can add more folders of avatars in the Images/Avatars/Gallery/ directory. To enable the user to use them, select Enable gallery avatars. Using remote avatars or allowing the uploading of avatars carries a small security risk, but the benefits encourage individuality and can help to maintain your memory usage.
- Admin Email Address - This will go out on all automatically distributed mail from your site. It may be worth using specific address purely for this purpose.
- Use SMTP Server for email - You may not need to set up this as most servers will send out mail without external SMTP. It is worth testing first to see if you need this to be set.
- PayPal IPN Settings - IntegraMOD 1.4.0 onwards, has greater integration with PayPal to promote your site funding. Take care that you set these correctly. And feel free to share the wealth this way, if you so wish :)
Configuration + Another Important section
Here you will be able to set the default values for users and override the if need be. When a category has override selected, the user will not be shown the options that have been withheld.
- Registration information - Items marked with a red star will be compulsory for the user to submit information.
- Profile Settings - Internationalization - Here you can restrict the user to maintain a board system if appropriate. e.g. If a Forum is set in one country alone then only one timezone may be needed.
- Profile Settings - Privacy - As Board Founder, you have a duty to protect the information, unsuspecting persons may give. If you or the user selects not to display their personal information, themselves and admins only will be able to see it, with the addition of nominated friends if selected.
- Profile Settings - Reading - Please make sure you have 'Display images' set as Yes, otherwise unregistered visitors will not automatically be able to see images in forum posts
Further setting below this help you manage your site behaviour.
Hacks/Mods List Please visit and see over 100 MODs added into IntegraMOD to make it 'The Best Pre-MODded phpBB Package Available'. If Mod owners find their MODs have not been correctly labelled or updated please contact the IntegraMOD Team.
Meta Tags + Here is an opportunity to add search engine information to hopefully boost your position in the global ratings.
Optimize DB Enable Cron and set to how often you'd like your DB to be optimized, this will help  make your forum run smoother.
Welcome PM Here you can edit the Private Message each new user receives.
Qbar This is the hub of you site navigation. You have a variety of ways to show links on your site and Qbar also can set permissions as to who sees those links.
Rating System In order to allow users to rate post and images you need to set 'Rating system active' to Yes. Then click Update configuration at the base of this block.
Restore Database It is worth noting this tab - if you have previously backed up your database and your site has need to return to a previous state - you can upload you database backup here.
Group Admin
You can set specific groups here to allow private forum, or groups when user's have given a set number of posts
IM Portal
Masterdavid created a superb system which allowed phpbb owners a portal system that could offer a highly customisable web system. This offers a continuity of layout throughout the site, and new pages can easily be constructed whilst maintaining the overall professional presentation. Integrating this with with the other phpBB based MODs make IntegraMOD a formidable package.
Blocks Management - Selecting IntegraMOD Default, you are shown the main block structure for the majority of your site. Caution should be taken here as Deleting a block may delete it forum-wide.
- Edit - when clicked on a specific block line, will take you to the configuration of that block. You can change the block title here, and you may wish to change the names of 'IntegraNews' For you portal news column,  and 'Welcome to IntegraMOD' to something more pertinent to your site. If you do change the title , you may need to set the Localize Titlebar to 'No'.
- Most of the blocks have preset code in the site files, however you will see that the 'Welcome to IntegraMOD' block has content you can easily change, making its either BBcode based or HTML.
- It is worth noting - the cache for the following blocks must be set to No (disabled);- Board Navigation, IntegraNews, Recent Topics and Who is Online.
Delete Cache Files This enables the portal to take the latest information from the Forum to be displayed around the site. I would be would clicking this after you have updated you Blocks management section.
Portal Configuration - Default Portal Page - Although not at first, you may build a portal page that you wish to be the home page of the site other than the current one. If so you can set this here.
- Links -> Style - You can decide whether you wish scrolling or static links on your portal page. the same applies for the Recent Topics block.
- Search option - Forum text - The portal search block at the bottom of the left hand column, can search your site, or send a search request to Google. Change IntegraMOD to a short tile for your own site.
- Display users who didn't visit - Although not initially, as you site grows you may want to remove the tick in this box.
Knowledge Base
The Knowledge Base is an incredibly useful way of collecting and displaying articles of interest to your site visitors. Users can post articles which can be approved before being displayed in the portal.
Configuration - KB Moderator Group - It is worth setting this to Admin
- Forum ID - it is worth making a forum specifically for knowledge base articles, and then reference it here.
- Allow Ratings and Allow Comments, both add extra feedback to the article creators.
Donors Admin
When you set up you PayPal IPN and configure this section, your Donations Block will link users to PayPal in the hope they like your site :)
Languages
When you have more than one language to offer users, this can offer greater manipulation of your lang files.
Links
You can configure how users can offer links to outside your site an upload images to further highlight them.
News Management
Your portal contains a comprehensive news display system, allowing topics to be shown categorised, in archive form or as latest news from from the site. Also included is an RSS News feed system to allow news from your forum to be displayed on others' sites without any further input from you!
Categories You will need to add at least one category in order for your News system to start functioning. Click on Add New News Category, where you will need select an image an give the category a title. You can upload your own images to the images/news/ directory.
Configuration - News Mod Base URL - set this to the exact URL of your portal.php (e.g. http://www.mysite.com/forum/portal.php)
- News Mod Index File - make sure this is portal.php
- Enable RSS Syndication - although very useful, you may not wish your news so easily accessible by others. Is so, set this button to 'No'
P.C.P.
The Profile Control Panel is exactly what it says, but adds more functionality, information and individuality to your forum. By defining more fields that relate to users, more enjoyment and understanding takes place. Initially you will probably not wish to touch this. However there is now the PCP Wizard, which can help the intermediate Board Founder flourish.
Photo Album
Smartor's Photo album is a useful feature on the site. It provides Personal and Public Galleries that can also update the homepage and a part of the Users Profile. You can set a watermark to be used to protect the copyright of the images from your site.
Categories You will need to make at least one Category in order to use the public Galleries. You can also set user permissions as to who can upload and view these galleries.
Clear Cache It is worthwhile clearing your thumbnail cache, especially after every time you change any album settings.
Configuration You can set up maximum sizes for downloads here, with the options for multiple uploads.
- Maximum number of files user can upload at a time - Initially set at one you can allow many uploads from one entry point.
- Optimize for the version of GD - Most of the latest versions of Linux have GD2 which we have placed by default. You can tell your GD version by looking in ACP/Security/Info:GDlib. It is important to get this correct or your images will be poor.

Prillian
Prillian is a cross some where between a chat system and a PM system. It can give you instant message that pop up in front of you to Keep you on the story. Some would rather disable it. Here are the settings to disable Prillian:-
Configuration Enable Instant Messaging System No
Enable Network Messaging system No
Override user settings Yes

Pseudocron
Pseudocron is you IntegraMOD internal clock. It Keeps ticking and sets up the sending out of Digests to users. This need to be set to On to promote better automation.

Statistics

Here you can configure how you would like to see your Statistics page, and add more or remove items.
Styles Admin
This is partly left over from the phpBB days, but of importance is Banner Admin where you can setup rotating banners on your site, and specify, the complete running of a banner ad campaign
Subscriptions
Organise subscriptions to individual forums though this control panel

Tools

PHPInfo This is a very useful diagnostic page if faults arise and

User Admin

This final section really helps to manage you users and any bad influences too.
Admins & Mod's A useful quick reference as to how many users hold levels of authority. Remember this should also concur with the Security MOD/Special Settings.
Ban Control You can ban or un-ban individuals, IP Addresses or whole IP domains. You will also find that the security mod will add IPs and Users if there are hacking attempts or the user enters the wrong password too many times.
Disallowed Names Enter names here you do not wish to be used one the forum as they are either rude, or are close to other members of the forum.
Junior Admin Sometime you wish to give members Admin responsibilities but not the whole show. You can give Junior Admins as much or as little power as you wish. To make a junior admin, find their name in the Junior admin user list. Then click Edit Modules beside their name. You will now see every admin function available. Simply tick the boxes to which powers you wish the person to have. No Junior admin or admin can change who the board founder is on your board but caution should still be observed as they may have access that could damage the boards reputation.
Manage On this section, enter a user's name and you will be able to amend any aspect of that persons profile, including their password, username, and points accumulated.
Permissions Here you can give individual users certain rights to particular forums. You must remember that if you make someone and admin or a moderator of a forum, they will not be able to access the forum until you have entered a minimum number of admins or mods in the Security module.
If You wish to give someone full Admin access to your site, you enter their username here, and then at the top of the page is a menu. Change their status from User to Administrator, and submit. Again Security restrictions apply.
Points Configuration You can give users points for the number of posts and replies they have made. These points can be named whatever you like and can be awarded via posting, user to user donation, paypal donation, Admin donation (where points can also be deducted) and through additional mods where points can be one or used as payment.
Private Messages Here you have access to every message that has been sent and receive until its owner has deleted it. A point of not that before a pm has been read, it is only one, but is duplicated once read, one for the sender and one for the recipient. This is where admin discretion needs to be the better part of Valour if you dare read anyone else's PM!
Prune users You can automatically set limits for users to be pruned off the forum, or you can manage them by hand. Caution, some may be regular visitors, just not good posters.
Ranks When Rank images are uploaded to your server they appear in the dropdown list for you to select a hierarchy of staff and users. The may be set manual, or by number of posts they have made. Do note that changing someone the Site Admin here does not give any admin level permissions.
User list This is a comprehensive list of all your users, able to be sorted in most formats. Selecting Open will also allow you to manage, set permissions or contact them via e-mail or PM. If you tick the box beside them you will find a dropdown menu at the bottom of the table where you can Ban, add to a group, activate, deactivate or delete those you have selected.
Inactive Users A colourful page devoted to the most lethargic users!


Once you've done these then feel free to familiarise yourself with the rest of the Admin controls you have at your command.
